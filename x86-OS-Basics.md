# 实现本操作系统的基础知识
# 常见段寄存器
CS  代码段
DS  数据段
ES  附加数据段
FS  扩展段
GS  扩展段
SS  栈段

# 源地址寄存器和目标地址寄存器
esi  源地址寄存器
edi  目标地址寄存器
ecx  拷贝次数寄存器
使用案例
mov esi, 0x10000   ; 源地址给 ESI
mov edi, 0x20000   ; 目标地址给 EDI
mov ecx, 100       ; 拷贝100字节
rep movsb          ; 自动循环拷贝

# 常用寄存器
eip  指令寄存器 CPU 下一条要执行的指令的内存地址
esp  栈指针寄存器
ebp  基础指针寄存器
eflags  状态标志寄存器,里面存的是 CPU 刚执行完指令后的各种 “状态、结果、开关”
eax,ebx,ecx,edx  通用寄存器
ax,bx,cx,dx  通用寄存器exx的低16位
al,bl,cl,dl  ax,bx,cx,dx的低8位
ah,bh,ch,dh  ax,bx,cx,dx的高8位

# 汇编语法
int $0x80      正确：触发 0x80 号中断(AT&T 汇编语法)
int 0x80       错误：去访问内存地址 0x80 的值，然后当中断号(AT&T 汇编语法)

GCC内联汇编格式
__asm__ __volatile__(
    "指令"
    : 输出部分 (写结果)
    : 输入部分 (给数据)
    : 破坏部分
);
# 计算机组成原理部分知识

## 一、数制与编码（基础开篇）

- 原码、反码、补码（重中之重）: 有了补码不用减法器

- 定点数、浮点数表示（IEEE754 单/双精度）:简单例子 2.5是在计算机里是如何表示的，以为为什么0.1无法在计算机里准确表示

- BCD码、ASCII、奇偶校验、海明校验码

## 二、运算器与算术逻辑单元 ALU

- 加法器：串行进位、先行进位（超前进位）

- 加减乘除硬件实现（原码/补码运算）

- ALU 结构、标志位（进位、溢出、零标志、符号位）
  
- CPU整体架构 :ALU,寄存器,cache缓存,CU控制器等等

- 移位运算、桶形移位器

## 三、存储器层次体系（超级重点）

- 存储器分类：ROM/RAM、SRAM/DRAM :四种的区别和分别应用在哪部分

- 主存、Cache、虚拟内存 层级关系

- Cache 映射：直接映射、全相联、组相联

- 替换算法：LRU、FIFO

- 主存与Cache 地址划分：标记、组号、块内偏移

- 虚拟内存、页式存储基本概念（衔接操作系统）

- 内存条位扩展、字扩展

## 四、指令系统与寻址方式

- 机器指令格式：操作码 \+ 地址码

- 指令分类：数据传送、算术逻辑、控制转移

- 寻址方式：立即寻址、直接、间接、寄存器、变址、相对寻址

- 精简指令集 RISC / 复杂指令集 CISC 区别

## 五、CPU 架构与控制器（全书最难）

- CPU 内部组成：运算器、控制器、寄存器组、程序计数器 PC、指令寄存器 IR

- 指令执行流程：取指 → 译码 → 执行 → 访存 → 写回

- 时序控制：时钟周期、机器周期、指令周期

- 控制器设计：组合逻辑控制器、微程序控制器

- 微指令、微程序、微地址形成

## 六、流水线技术（进阶重点）

- 指令流水线原理

- 流水线冒险：结构冒险、数据冒险、控制冒险

- 解决方案：插入停顿、数据旁路、分支预测

- 超标量、超线程基本概念

## 七、总线系统

- 系统总线分类：数据总线、地址总线、控制总线

- 总线仲裁、串行/并行总线

- 总线时序、同步/异步通信

## 八、输入输出 I/O 系统

- I/O 编址：统一编址、独立编址

- 四种 I/O 控制方式：
        

    - 程序直接控制（轮询）

    - 中断驱动

    - DMA 直接存储器存取

    - 通道方式

- 中断原理、中断响应、中断处理流程

- DMA 工作过程、用途
